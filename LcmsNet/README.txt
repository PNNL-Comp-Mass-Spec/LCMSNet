The Documentation directory contains documentation reguarding the lcmsnet program.

The lcms directory contains the main program solution.

The lib directory contains all needed support libraries created by the SDK solution projects.  Additionally, this directory contains a .gitignore file that guarantees that while the directory exists within the repo, none of the files are tracked.

The SDK directory contains the SDK solution and associated projects, including the lcmsnetSDK, fluidicsSDK, fluidicsSimulator, fluidicsPack, and PDF generator. It also contains the solution for the Document Generator upon which the PDF generator is built.
