To build a deployment-ready library, build the PNNLRelease configuration, which will copy the .dll file to the \lib directory.

SQLite is using package System.Data.SQLite.Core, which corresponds to files:
* sqlite-netFx40-static-binary-Win32-2010-1.0.x.0.zip
* sqlite-netFx40-static-binary-x64-2010-1.0.x.0.zip

from https://system.data.sqlite.org/index.html/doc/trunk/www/downloads.wiki
